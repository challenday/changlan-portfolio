# Rental Smart

## Overview
Rental Smart is a Property Management solution designed in Microsoft Power Platform and Dataverse to support lease tracking, maintenance requests, vendor coordination, rent workflows, and operational reporting.

## My Role
- Business Analysis
- Requirements gathering
- Data model design
- User story creation
- UAT support
- Documentation

## Tools
- Dynamics 365
- Power Apps
- Power Automate
- Dataverse
- Power BI

## Key Features
- Lease tracking
- Maintenance request workflow
- Vendor assignment
- Rent reminder process
- Dashboard reporting
